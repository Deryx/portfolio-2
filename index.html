about

I have been enamored with web programming since I discovered that web pages were created using HTML. I promptly purchased the HTML For Dummies book and within two weeks built my first website. The site was a personal training site made interactive with Perl. I was hooked! I thought "how could I do this for a living?"

For my next project, since I was an Associate Director of Financial Aid, I decided to make a financial aid estimation program using HTML and a newly learned technology, classic ASP. The project was a success! Since the college did not yet have a web presence and I had decided to leave, the college created the Web Developer position for me. Thus I had become a professional web programmer!

This has resulted in an almost twenty year journey that started with HTML and Perl and has progressed to the addition of CSS, JavaScript and its frameworks, jQuery, ASP.NET, C#, and SQL. Web development is never stale as new technologies seem to be created everyday. I enjoy learning and employing these technologies to create clean, attractive, and dynamic web sites and applications for whomever needs them. May I work with you?
